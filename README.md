# 🐶 Dog API Search

Projeto desenvolvido para a sprint de **Consumo de API com JavaScript**.
A aplicação permite pesquisar raças de cachorros e exibir informações e imagens utilizando a API pública **The Dog API**.

---

## 📌 Objetivo

Consumir dados de uma API utilizando **programação assíncrona em JavaScript**, interpretar os dados em **JSON** e exibi-los dinamicamente na interface da aplicação.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* The Dog API

---

## ⚙️ Funcionalidades

* Pesquisa de raças de cachorros
* Consumo de API utilizando `fetch`
* Uso de **async/await**
* Manipulação do **DOM**
* Exibição dinâmica de dados
* Tratamento de erros caso a API não responda ou não encontre resultados
